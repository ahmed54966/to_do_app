📝 To-Do List App - Flutter & Firebase
A powerful and efficient Task Management application built with Flutter and Firebase. This app allows users to organize their daily tasks, set deadlines, and sync data in real-time across multiple devices using Google's Firebase services.

✨ Features
Authentication: Secure user Login and Registration flow.

Real-time Database: Tasks are stored and synced instantly using Firebase Firestore.

Task Management: Create, Read, Update, and Delete (CRUD) tasks effortlessly.

Calendar Integration: A horizontal calendar view to manage tasks by specific dates.

Swipe to Action: Interactive UI allowing users to swipe to delete or manage tasks.

User Settings: Change app language and manage user profiles.

Splash Screen: Professional branded entry animation.

🚀 Tech Stack
Frontend: Flutter (Dart)

Backend: Firebase Authentication & Firestore (NoSQL Database).

State Management: ( Provider.

UI Components: Custom buttons, TextFields, and Date Pickers.
